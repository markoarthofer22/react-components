# Breadcrumbs

Breadcrumbs component for everday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Breadcrumbs from '@markoarthofer22/react-components.breadcrumbs';
// OR
import { Breadcrumbs } from '@markoarthofer22/react-components';
```
