# SweetAlert

Wrapper for popular alert lib - reactivized :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import SweetAlert from '@markoarthofer22/react-components.popup';
// OR
import { SweetAlert } from '@markoarthofer22/react-components';
```
