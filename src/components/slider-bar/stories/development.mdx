# SliderBar

SliderBar component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import SliderBar from '@markoarthofer22/react-components.slider-bar';
// OR
import { SliderBar } from '@markoarthofer22/react-components';
```

<Canvas withSource={'open'}>

This SliderBar is not dependable on `input` component so that makes it support all of the browseres.
By the default you need to pass `initialValue` prop that specifies where slider starts. All of the other props are optional

If you want you can format your inputs by passing `formatFn` prop (`number => number.toFixed(0)` by default). Also if you want to retrive value from the slider you can use `onChange` prop
