# Grid component

Custom grid component that uses Flex for positioning.

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import { Grid, Col, Row } from '@markoarthofer22/react-components.grid';
// OR
import { Grid, Col, Row } from '@markoarthofer22/react-components';
```
