# StyledList

List component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import {
    StyledListItem, StyledList
} from '@markoarthofer22/react-components.styled-list';
// OR
import { StyledListItem, StyledList } from '@markoarthofer22/react-components';
```

Accordion only accepts `React.Component` of type `StyledListItem`. Wrap your content inside of the component (for state managment). Be aware, you can pass only **ONE** element inside of the `StyledListItem`. If you want to pass more then one, wrapp children with `<React.Fragment />`
