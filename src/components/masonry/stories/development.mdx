# Masonry Layout

Create simple MasonryLayout component

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import MasonryLayout from '@markoarthofer22/react-components.masonry-layout';
// OR
import { MasonryLayout } from '@markoarthofer22/react-components';
```
