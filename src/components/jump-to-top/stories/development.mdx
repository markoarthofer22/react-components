# JumpToTop

JumpToTop component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import JumpToTop from '@markoarthofer22/react-components.jump-to-top';
// OR
import { JumpToTop } from '@markoarthofer22/react-components';
```

`visibleFrom` prop should be in `px` format. `targetElement` is optional but it should be passed for better accuracy. If not, component will target top of the window

Also you can pass `icon` prop (`React.ElementType | React.ComponentType`) for custom button icon

<div style={{ marginBottom: '50px' }} />
