# Avatar

Avatar are component that show an array of values as small badges

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Avatar from '@markoarthofer22/react-components.avatar';
// OR
import { Avatar } from '@markoarthofer22/react-components';
```

<Canvas withSource={'open'}>
    <Story id='designs-atoms-avatar--default' />
</Canvas>

Avatar has no **required** dependencies except `children` property. It has 3 sizes:
`small | normal | big`. Default is `normal`. Check the table at the bottom to see
all of the available props.
