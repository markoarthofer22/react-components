# Accordion

Accordion component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import {
    Accordion,
    AccordionItem,
} from '@markoarthofer22/react-components.accordion';
// OR
import { Accordion, AccordionItem } from '@markoarthofer22/react-components';
```

    <Story id='designs-atoms-accordion--default' />
</Canvas>

Accordion only accepts `React.Component` of type `AccordionItem`. Wrap your content inside of the component (for state managment). You must pass `title` prop to `AccordionItem`. Content goes trough `children` prop. Be aware, you can pass only **ONE** element inside of the `AccordionItem`. If you want to pass more then one, wrapp children with `<React.Fragment />`
