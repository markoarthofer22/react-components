# Tooltip

Tooltip component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Tooltip from '@markoarthofer22/react-components.tooltip';
// OR
import { Tooltip } from '@markoarthofer22/react-components';
```
