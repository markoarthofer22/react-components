# Popup

Popup component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Popup from '@markoarthofer22/react-components.popup';
// OR
import { Popup } from '@markoarthofer22/react-components';
```

<Canvas withSource={'open'}>
    <Story id='designs-atoms-popup--default' />
</Canvas>

Popup state is not controled from the component itself. You need to pass `closePopup` prop that will handle the change.

Also you can pass `icon` prop (`React.ElementType | React.ComponentType`) for custom closeIcon
