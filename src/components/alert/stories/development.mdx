# Alert

Alert are component that show an array of values as small badges

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Alert from '@markoarthofer22/react-components.alert';
// OR
import { Alert } from '@markoarthofer22/react-components';
```

**Alert => still in development. Selfclosing feature is missing.**

You need to pass `show` prop to open Alert. State is managed outside of the component.

This alert is guided by Material Alert component. Check the table for all of the available props. Icon is `boolean` by default, but you can pass any `React.ReactNode` element. If `icon` is `true` default Icon will render.

You can pass `backgroundColor` prop if you want to manualy change background Color of the element.
