# QuantityInput

QuantityInput component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import QuantityInput from '@markoarthofer22/react-components.quantity-input';
// OR
import { QuantityInput } from '@markoarthofer22/react-components';
```
