# Side Nav

Side navigation animated with [Framer Motion](https://www.npmjs.com/package/framer-motion)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import SideNav from '@markoarthofer22/react-components.dialog';
// OR
import { SideNav } from '@markoarthofer22/react-components';
```
