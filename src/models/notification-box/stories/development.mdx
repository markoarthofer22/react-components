# NotificationBox

NotificationBox component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import NotificationBox from '@markoarthofer22/react-components.notification-box';
// OR
import { NotificationBox } from '@markoarthofer22/react-components';
```
