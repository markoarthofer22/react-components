# Dialog

Dialog component for everyday usage :)

### Usage and import

You can import it in two ways: - as a standalone component - from the whole package

```jsx
import Dialog from '@markoarthofer22/react-components.dialog';
// OR
import { Dialog } from '@markoarthofer22/react-components';
```
